# La Providencia — página institucional

Página one-page de La Providencia I y II construida con Vite, JavaScript y CSS.

## Desarrollo

```powershell
npm install
npm run dev
```

## Producción

```powershell
npm run build
npm run preview
```

## Estructura

- `assets-source/`: imágenes originales antes de optimizar.
- `docs/`: plan del proyecto y registro de cambios.
- `public/img/`: imágenes publicadas y optimizadas.
- `public/video/`: videos utilizados por la página.
- `src/data/`: datos fuente del catálogo.
- `src/js/`: comportamiento y animaciones.
- `src/styles/`: estilos del sitio.

La explicación consolidada de las decisiones y cambios está en
`docs/CAMBIOS_IMPLEMENTADOS.md`.
